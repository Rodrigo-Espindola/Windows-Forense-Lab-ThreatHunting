# Windows-Forense-Lab-ThreatHunting
Laboratório prático de Blue Team e Threat Hunting em Windows. O projeto simula o dia a dia de um analista focado em detectar persistência de malwares e rastrear comportamentos anômalos no presente, lidando com o desafio real do tempo de retenção de logs e expiração de evidências históricas através do endurecimento do Sysmon e Event Viewer.
